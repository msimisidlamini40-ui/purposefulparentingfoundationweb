# Purposeful Parenting Foundation

A landing page for the **Purposeful Parenting Foundation**, a South African Non-Profit Organization (NPO) dedicated to interrupting cycles of generational pain and raising the next generation with intention.

## 🌟 Overview

This project is a modern, responsive website designed to showcase the foundation's mission, workshops, and resources. It features a sophisticated earth-toned aesthetic and smooth interactive elements to engage users effectively.

## 🚀 Key Features

- **Responsive Design**: Optimized for all screen sizes, from mobile to desktop.
- **Scroll Animations**: Uses `IntersectionObserver` for elegant fade-in effects as users navigate.
- **Dynamic Counters**: Animated statistics to highlight the foundation's impact.
- **Interactive FAQ**: A clean accordion-style FAQ section for common inquiries.
- **Smooth Navigation**: Custom smooth-scroll behavior for a seamless user experience.
- **Contact Form**: A functional contact interface with basic validation.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better accessibility and SEO.
- **CSS3**: Custom properties (CSS variables) for a consistent theme and responsive layout.
- **JavaScript (ES6+)**: Custom scripts for animations, navigation, and form handling.
- **Google Fonts**: Utilizing 'Playfair Display' for headers and 'DM Sans' for body text.

## 📖 About the Foundation

The Purposeful Parenting Foundation provides workshops, support circles, and resources like the book *Parenting With a Purpose* by Pastor Vusumzi Mthembu. Their goal is to empower parents and guardians to raise children in a healthy, purposeful environment.
